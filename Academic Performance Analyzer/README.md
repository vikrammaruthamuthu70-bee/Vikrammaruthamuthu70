# Academic Performance Analyzer

A simple HTML/CSS/JavaScript app to analyze student performance for:

- UG Arts & Science (3 years / 6 semesters)
- UG Engineering (4 years / 8 semesters)
- PG (2 years / 4 semesters)

## Features

- Dynamic semester selection
- Subject-level marks, credits, and grade calculation
- Automatic SGPA and CGPA calculation
- Performance summary with pass/fail counts
- Charts for semester SGPA and grade distribution

## Run locally

Open `index.html` in your browser.
